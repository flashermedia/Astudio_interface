Title: Astudio_Interface
Description: This program allows ATMELS AStudio to program an AVR chip. It simulates an STK500 or JTAGICEMk2 board. The program allows a user to learn about 
JTAG interfaces and to see JTAG signals and state changes. So with Astudio a serial cable
one or two PC's an AVR board and a printer interface programming of ATMEL chips is possible from AStudio. Please read Readme.html directions on usage source code etc.
